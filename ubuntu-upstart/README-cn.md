# 什么是Upstart?

Upstart 是一个基于事件的传统init 守护进程的替代实现,凭借这种方法,在多种类unix系统启动时,执行一些任务.它是由Canonical公司前雇员Scott James Remnant编写.

> [wikipedia.org/wiki/Upstart](https://en.wikipedia.org/wiki/Upstart)
