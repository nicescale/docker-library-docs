# `ubuntu-debootstrap`

这个镜像是对当前支持的ubuntu发行版,运行`debootstrap --variant=minbase`命令产生的.它不是官方版本,不被Canonical以任何形式提供支持.请查看[`ubuntu`](https://registry.hub.docker.com/_/ubuntu/),了解受到Canonical支持的官方ubuntu镜像.
